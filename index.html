How to use this tool

1. Enter your processes

Type Arrival, Burst, and Priority values directly into the table. Use Add Process for a blank row, Generate Random for sample data, or check rows and press Delete Selected to remove them.

2. Pick an algorithm

Choose Round Robin, Priority, or SJF from the dropdown. Round Robin also needs a Time Quantum — the fixed slice of CPU time each process gets per turn.

3. Calculate

Press Calculate to run the simulation. The Statistics Dashboard, Per-Process Metrics table, and Gantt Chart all update together.

4. Watch it play out

Press Play on the Gantt Chart to animate a playhead across the timeline. Use the speed buttons to slow down or speed up the animation, and Restart to replay it.

5. Compare algorithms

Press Compare to run all five algorithms against your current process list and see which one produces the lowest average waiting and turnaround time.

Reading priority

Priority is a plain number: 1 is the highest priority, and larger numbers run later.
